The "Pick And Plate.conf" file contains the QSettings for the application as of 08/19/2019.
This file must be placed in "~/.config/OSU SARL/" to apply these settings at application launch.

The tinyg_configuration file is a raw dump of the tinyg motion controller's settings. 
To re-apply them, follow the tinyg confiugration guide and set the appropriate fields to the values in this file.
